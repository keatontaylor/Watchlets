What are Watchlets?

 

Watchlets displays user-selected data on the Apple Watch; add watchlets to display RSS feed items, stock data, or even your shopping list. All you need is a properly formatted JSON object with a key value pair containing your data. Add the URL of the JSON resource on your phone, and your Watchlets app will display the returned JSON object on a watchlet page.

 

If your URL supplies multiple JSON objects in an array, each will be visible on a new watchlet page; just swipe horizontally to scroll between pages. If you add additional URLs, the corresponding watchlet pages will be added to the end of the existing list of pages.

Launch the Watchlets app on your Apple Watch and swipe horizontally through the watchlet pages.

 

Refresh

Force Touch on a watchlet page to present refreshing options.

 

Watchlets API

Watchlets supports four JSON keys, formatted in a JSON message as below:

 

{
  “content”:
    [
      {
         “id”: “5001”,
         “title”: “Watchlets!”,
         “text”: “An easy way to see your data.”,
         “image”: “http://example.com/image.png”
      }

    ]
}

 

id- allow refreshing this watchlet individually (optional)

 

title- short title displayed above the body text of the watchlet (optional)

 

text – text displayed in the body of the watchlet. Watchlet supports most standard escape sequences, such as “\n” for newline. (optional)

 

image - a URL pointing to a PNG image. Watchlet works best with 120x120px PNG images for the title, and 500x500px PNG images for the image Watchlet. (optional)
